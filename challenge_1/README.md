# KPI Bonus Calculator

This Python program calculates the KPI bonus based on the user's salary and bonus percentage.

## 📌 Formula
The bonus is calculated using:
KPI = 1000 + salary * bonus_percentage


## 🧩 How to Run the Program
1. Make sure you have **Python 3** installed.
2. Save the script as `kpi.py`.
3. Open a terminal in the folder where the file is saved.
4. Run the program with:
python3 kpi.py


5. Enter:
   - Your name  
   - Your salary (decimal allowed)  
   - Your bonus percentage (decimal allowed)
## 📝 Example
Input:
Name: Luciano
Salary: 5000
Bonus: 1.5
Output:
Olá Luciano, o seu bônus foi de 8500
