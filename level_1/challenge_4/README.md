# Challenge 4: Data Manipulation with Lists and Dictionaries

This folder contains exercises focused on manipulating data structures in Python, specifically lists and dictionaries. These exercises build on basic data types and introduce common data processing patterns.

## File

### `exercises_4_1.py`
Contains exercises for:

- Duplicate removal from lists
- Custom sorting of dictionary lists
- Data aggregation (averages)
- Splitting data into groups (even/odd)
- Updating data in dictionary lists
- Merging dictionaries
- Filtering dictionary data
- Extracting keys and values
- Character frequency counting

## Exercise Summary

1. **Duplicate Removal**
   - Remove duplicates from a list of emails using sets.

2. **Custom Sorting**
   - Sort a list of people dictionaries by name.

3. **Data Aggregation**
   - Calculate the average of a list of numbers.

4. **Splitting Data into Groups**
   - Separate even and odd numbers into different lists.

5. **Data Update**
   - Update the price of a specific product in a list of product dictionaries.

6. **Dictionary Merge**
   - Combine two dictionaries into one.

7. **Filtering Data in a Dictionary**
   - Filter products with positive stock quantities.

8. **Extracting Keys and Values**
   - Create separate lists for dictionary keys and values.

9. **Character Frequency Count**
   - Count the frequency of each character in a string using a dictionary.

## Learning Objectives

- Work with Python's built-in data structures (lists, dictionaries, sets)
- Apply list comprehensions for filtering and transformation
- Use dictionary methods for data manipulation
- Implement basic data aggregation and analysis
- Practice common data processing patterns

## How to Use

1. Open `exercises_4_1.py`
2. Uncomment the block for the exercise you want to run
3. Execute the file:
   ```bash
   python exercises_4_1.py
   ```

## Notes

- All exercises are written as commented code blocks for step-by-step practice.
- Uncomment and test one exercise at a time to avoid overlapping output.
- These exercises introduce fundamental data manipulation techniques used in data processing and analysis.
