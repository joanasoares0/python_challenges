🐍 Python Challenges Repository

A collection of Python challenges solved for learning, practice, and skill improvement.
Each challenge is organized in its own folder, containing the problem description, the solution, and any additional notes or files needed.
The exercises are arranged with progressive difficulty so you can train from basic concepts to more advanced problems.

📁 Repository Structure
The repository is organized as follows:

python_challenges/
│
├── challenge_01/
│   ├── description.md
│   └── solution.py
│
├── challenge_02/
│   ├── description.md
│   └── solution.py
│
├── challenge_03/
│   ├── description.md
│   └── solution.py
│
└── ...
Each folder represents a standalone challenge.

🎯 Purpose
This repository serves to:
Practice Python through real exercises
Track progress over time
Organize solutions in a clean and reusable way
Build a personal library of solved problems
Improve problem‑solving and coding style

🚀 How to Use
Clone the repository:

Code
git clone https://github.com/joanasoares0/python_challenges.git
Navigate to any challenge folder:

Code
cd python_challenges/challenge_01
Run the Python solution:

Code
python3 solution.py


📌 Notes
New challenges will be added over time.
Feel free to explore, learn, and adapt the solutions.
